100 squares
===========

Another grid-based game.

Build
-----

You need [Node](http://nodejs.org/) and [Grunt](http://gruntjs.com/getting-started) to build.

Enter:
```
grunt build
```

If you want to generate a graph of dependencies, enter:
```
grunt madge
```
